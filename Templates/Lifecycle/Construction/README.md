# Construction/
Tactical artifacts created during active development. Starts empty, grows organically.
## Subdirectories
### Test-Plans/
Proposed test cases for features. Populated when testing is planned.
Template: `Test-Plan-{feature}.md`
### Design-Decisions/
Technical decisions with context and rationale. Populated when architectural choices are made.
Template: `Decision-{NNNN}-{title}.md`
### Sprint-Retros/
Retrospective summaries from sprints. Populated after `/microsprint close`.
Template: `Retro-{sprint-name}.md`
### Tech-Debt/
Identified technical debt for later. Populated when work is deferred.
Template: `Debt-{title}.md`
